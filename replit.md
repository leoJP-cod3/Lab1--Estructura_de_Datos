# Ejercicios sobre el uso de Listas y Pilas

## Overview
A C programming exercise lab for students, covering linked lists (ArrayList TDA) and stacks (Stack TDA).

## Project Structure
- `exercises.c` — Student exercise file (the only file students should modify)
- `test.c` — Test suite (auto-grading)
- `test.sh` — Build and test runner script
- `arraylist.h` / `arraylist.c` — ArrayList TDA implementation
- `stack.h` — Stack TDA header
- `log` — Tracks submission history

## Running Tests
Open the Shell and run:
```bash
bash test.sh
```

## Workflow
- **Start application**: Runs `bash test.sh` (console output)

## Notes
- No frontend or backend web server — pure C project
- Only `exercises.c` should be modified by students
- GCC 14.3.0 available in the environment
